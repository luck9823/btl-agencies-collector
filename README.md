# BTL & Marketing Agencies Data Collector

Сбор базы российских BTL и маркетинговых агентств с выручкой >= 200 млн руб.

## Результат
- Файл: data/companies.csv
- Компаний: 50
- Фильтр: выручка >= 200 млн руб за 2023 год

## Запуск
pip install -r requirements.txt
python src/main.py

## Источники данных
1. РРАР-2024/2025 - рейтинг рекламных агентств России
2. AdIndex - обороты рекламных групп
3. АКАР DNMR - рейтинг digital агентств
4. РАМУ - Российская ассоциация маркетинговых услуг

## Подход
Использовали отраслевые рейтинги как seed-список. Для каждой компании собирали ИНН и проверяли выручку. Фильтрация: revenue >= 200 млн руб.

## Роль LLM
Claude использовался для нормализации описаний и документации.
